# Reflection on Control Statements Lab (BEE 208)

## Overview
This lab focused on applying **C++ control statements** (if/else, switch, loops, break, continue) to solve practical engineering problems. Each program simulated real-world scenarios such as fuse selection, motor safety checks, energy calculations, and sensor monitoring.  


Conditional Logic
  I learnt how `if` and `switch` statements can be used to make decisions in engineering systems. For example, checking whether a motor’s starting current exceeds safe limits or selecting the correct fuse rating for a circuit.  

Loops for Repetition  
  Using `for`, `while`, and `do...while` loops helped automate repetitive tasks like summing lighting loads, calculating averages, and performing repeated safety checks. This mirrors how engineers process multiple readings in real systems.  

Break and Continue 
  These statements taught me how to stop or skip processes when necessary. In engineering, this is similar to halting a test when unsafe conditions are detected or ignoring faulty sensor data to maintain accuracy.  

Practical Engineering Context  
  Each program connected directly to electrical and electronic engineering applications:
  - Fuse ratings → circuit protection  
  - Motor current checks → starter design  
  - Energy calculations → load analysis  
  - Sensor monitoring → fault detection and reliability  

Challenges
- Ensuring **consistent program layout** across all 20 questions was challenging at first, but adopting a uniform style (menu prompts, clear outputs, structured logic) made the code easier to read and debug.  
- Understanding how to translate engineering requirements into **logical conditions** required careful thought, especially for safety-related checks.  

Applications
This lab showed me how programming is not just abstract logic but a tool for solving **real engineering problems**:
- Automating repetitive calculations in energy systems  
- Designing safer circuits by embedding decision logic  
- Monitoring and controlling devices in automation systems  

Reflection
Overall, this lab strengthened my confidence in using **C++ control statements** to model engineering processes. It highlighted the importance of programming in modern engineering practice, where automation, safety, and efficiency depend on reliable logic. By mastering these concepts, I am better prepared to design systems that are both technically sound and practically useful.
